Java SSH 整合

  • SSH
Java

一.环境

配置相关

  • Mac OS X 10.12 macOS Sierra
  • JDK 1.7.0_79
  • SSH 版本
    struts-2.3.31-all
    spring-framework-3.2.0.RELEASE
    hibernate-distribution-3.6.10.Final

也可以根据需要选择最新的jdk1.8,同时把SSH各个版本升级到最新。

  • Eclipse 版本
    Eclipse Java EE IDE for Web Developers.
    Version: Luna Service Release 2 (4.4.2)
    Build id: 20150219-0600

二.创建Web项目

1.创建一个web项目(Eclipse EE 版)

如图创建一个Web工程，命名为S2SH。

2.导入Structs2中的包

解压struts-2.3.31-all.zip得到以下目录

struts2-blank.war 是structs2-blank的空工程，解压。copy WEB-INF/lib/下的所有jar包。

将copy的所有包放到S2SH工程中WEB-INF/lib/下。如下图

3.修改web.xml配置文件

在WEB-INF/下添加web.xml配置文件

4.添加struts.xml文件

在src中添加struts.xml 文件

5.导入spring的jar包

导入 spring-framework-3.2.0.RELEASE中所有*.RELEASE.jar 包
导入后如图，其中选择的部分是不必要的jar包，可以删除。

为了整合需要，我们还需要struts2-spring-plugin-2.3.31.jar 在struts-2.3.31/lib中。
同时要修改web.xml文件。

6.增加applicationContext.xml配置文件

在工程的src中添加applicationContext.xml配置文件,配置先关的bean以及依赖。

现在applicationContext.xml文件中我们没有进行配置任何bean，我们先导入hibernate的jar包，然后进行配置。

7.整合hibernate

在hibernate-distribution-3.6.10.Final下找到lib目录，如图，其中required目录是必须的包，optional是可选的包。

需要注意的是javassist-3.12.0.GA.jar这个jar包不用加，前面在添加structs中添加过。如果要用的注解的话还要导入hibernate-jpa-2.0-api-1.0.1.Final.jar

8.mysql数据库驱动

如果要连接mysql数据库，还需要添加mysql-connector-java-commercial-5.1.25-bin.jar包。
